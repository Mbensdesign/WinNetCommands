User Manual

Open the app by running netcmds.py.
Use the search box to find a command.
Use category buttons to filter commands.
Click a row to copy the command.
Type required values in the right column, such as host, IP, domain, username, or service name.
Click Run to execute the command inside the app.
Click PS to open the command in a PowerShell window.
Drag the separators to resize the command, description, and input columns.
Use STOP to stop a running command.
Use Clear Output or CLOSE to hide the output panel.
